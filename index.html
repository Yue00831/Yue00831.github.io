Yue Li

PhD Researcher · Computer Science & Cognitive Science

I am a researcher working at the intersection of computational linguistics, cognitive science, and large language models. My work investigates how humans and machines process language — from quantifier scope and animacy to working memory and probabilistic inference.

I am particularly interested in using behavioral experiments and computational modeling to probe the cognitive mechanisms underlying language comprehension, and in understanding how LLMs align with (or diverge from) human linguistic behavior.

Animacy and Orchestral Behavior in LLMs
Yue Li
Working Paper · 2024

This project investigates how animacy—the distinction between animate and inanimate entities—shapes language model behavior. Using controlled stimuli and behavioral probes, we examine whether LLMs exhibit animacy-driven processing biases analogous to those observed in human psycholinguistics.

Quantifier Scope, Reading Times & LLM Behavior
Yue Li
Working Paper · 2024

An empirical study comparing quantifier scope ambiguity resolution in human reading time experiments (SAP paradigm) with LLM predictions. We investigate whether surface-to-semantic mapping in LLMs mirrors psycholinguistic constraints from scopal processing.

Quantifier Scope Reading Times Semantics LLMs

Conditional Log-Probability as a Measure of Surprisal
Yue Li
Tool / Software · 2024

A Python toolkit for computing the conditional log probability of a target sentence given a context sentence. This measure of surprisal supports psycholinguistic research that uses LLM next-word probability distributions as proxies for human processing difficulty.

Working Memory and Language Processing
Yue Li
Ongoing · 2024–Present

This project explores the role of working memory capacity in sentence comprehension, particularly for syntactically complex structures. We examine individual differences in memory span and their interaction with grammatical complexity using behavioral and computational methods.
